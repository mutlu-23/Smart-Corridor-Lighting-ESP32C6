# Akıllı Koridor Aydınlatma Sistemi (ESP32-C6)

Bu proje, ESP32-C6 ile geliştirilmiş, hareket sensörü ve ışık algılama özelliklerine sahip bir LED aydınlatma sistemidir.

## Özellikler
- 🚨 Radar tabanlı hareket algılama
- 💡 Ortam ışığına göre otomatik parlaklık ayarı (TSL2561 sensörü)
- 🌈 5 farklı LED efekti (iz, nabız, gökkuşağı vb.)
- 📱 Web arayüzü üzerinden kontrol (AsyncWebServer)
- 🔄 OTA (Over-The-Air) güncelleme desteği

## Kurulum
1. Arduino IDE'ye ESP32-C6 desteğini ekleyin.
2. `Smart_Corridor_Lighting.ino` dosyasını açın.
3. `config.h` dosyasında WiFi bilgilerinizi girin.
4. Kodu ESP32-C6'ya yükleyin.
